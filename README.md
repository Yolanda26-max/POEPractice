# Cybersecurity Guardian Bot

A C# console application that educates users about cybersecurity awareness.
Built as part of the PROG6221 POE at Varsity College.

---

## Features

- Animated ASCII art header with cybersecurity theme
- Voice greeting using WAV audio playback
- Interactive chatbot with cybersecurity topics:
  - Phishing awareness
  - Password safety
  - Malware protection
  - Safe browsing
  - Privacy tips
- Quick security quiz with scoring
- Chat history tracking
- Colour-coded console UI with typing animations
- A Quiz

---

## How to Run

1. Clone the repository
2. Open `POEPractice.sln` in Visual Studio
3. Build the solution (Ctrl + Shift + B)
4. Run the project (F5)

---
